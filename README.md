# Morning Market Column AI Pro

Reuters / Bloomberg / Financial Times / Wall Street Journal / 日経などの公開見出し、市場データ、OpenAI APIを使い、毎朝の金融コラム・確認問題10問・株式シグナルを作るMVPアプリです。

## できること

- Reuters / Bloomberg / FT / WSJ / 日経の公開見出しを取得
- Nikkei Asia / WSJ Marketsなどの公開RSSも取得
- 主要指数・為替・金利・原油・金・BTCの市場データを取得
- ChatGPTで朝の金融コラムを生成
- コラムをもとに重要ポイント3つと4択問題10問を生成
- ウォッチリスト銘柄の機械的な「買い検討 / 様子見 / 売り・見送り検討」シグナルを表示
- 生成レポートを `reports/` にMarkdownとJSONで保存

## 重要な注意

このアプリは、記事本文の無断スクレイピングや再配信をしません。公開RSS / Google News RSSに出る「見出し・短いスニペット・リンク」を材料に、オリジナルの金融教育コラムを生成します。

商用化する場合は、Reuters / Bloomberg / FT / WSJ / 日経など各社のAPI契約・ライセンス・利用規約を必ず確認してください。

株式シグナルは教育目的の機械的な参考情報です。個別の投資助言・売買指示ではありません。

## 起動方法 Windows PowerShell

今のアプリ本体フォルダで以下を実行します。

```powershell
pip install -r requirements.txt
Copy-Item .env.example .env
uvicorn app.main:app --reload
```

ブラウザで開く：

```text
http://127.0.0.1:8000
```

## OpenAI APIを使う場合

`.env` をメモ帳などで開き、以下を設定してください。

```env
OPENAI_API_KEY=sk-xxxxx
OPENAI_MODEL=gpt-4.1-mini
```

API利用枠がない場合は、テンプレート版に自動フォールバックします。

## 主なAPI

- `/` : アプリ画面
- `/api/news` : ニュース取得
- `/api/markets` : 市場データ取得
- `/api/column` : 金融コラム生成
- `/api/quiz` : 確認問題10問生成
- `/api/signals` : 株式シグナル生成
- `/api/report?save=true` : 全部生成して保存

## v9 修正内容：実戦向けの価格設計

v7をベースに、株式シグナルの売買価格を現在価格そのままではなく、以下のルールで機械計算するように変更しました。

- 買い候補：現在値に飛びつかず、20日平均線・直近支持線・値幅をもとに押し目エントリー価格を表示
- 利確価格：損切幅に対しておおむねリスクリワード2倍以上を目安に表示
- 損切価格：直近支持線割れまたはATR風の値幅をもとに表示
- 売り候補：現在値ではなく、20日平均線や直近値幅をもとに戻り売りの売却価格を表示

これは教育目的の機械判定であり、利益を保証するものではありません。実際の売買では決算、開示資料、流動性、スプレッド、地合い、リスク許容度を必ず確認してください。
